What is CSS?
  1. Css stands for Cascading Style Sheets.

  2. Css describes how HTML elements are to be displayed on screen ,paper,or in other media.

  3. Css is not neccesary saves a lot of work. it can control the layout of multiple web pages all a once

  4. External stylesheet are stored in Css files,

CSS can be added to HTML documents in 3 ways:
  • Inline-by using the style attribute inside HTML elements l
  • Internal -by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS , is to keep the styles in external css file. however,in this tutorial we will use inline and internal and internal styles , because this is easier to demonstrate , and easierfor you to try it yourself.
